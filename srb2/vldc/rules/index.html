SRB2 VLDC #1 Rules

  • You must design the map to work with Demo 4 as a singleplayer level. (It must have 4 player starts, at least 1 deathmatch start, and an exit sector).
  • If you wish for your map to use a specific skybox from the game, then you may set the map to an ID other than MAP01.
  • You can use any custom textures / flats in your submission, but make sure to not leave any unused.
  • The maximum amount of maps you can add in one submission is 2, the second map is optional and must be a boss arena if you choose to add it.
  • Your map can use a cutom midi file.
  • Set the name of you wad to “(yourlevelname) - (yourusername)”.
