  • Zone Builder 2.9 (The version this configuration was based around)

    • KNOWN ISSUES AND FIXES

    ISSUE #1

  • Objects don't appear in the game automatically.

    • FIX
    Select every object in your map and toggle on the "Easy" "Hard" & "Multiplayer" Flags.

    ISSUE #2

  • Objects don't appear in normal mode when I enable the flags.

    • FIX
    Open the map in another configuration (Ex: Doom CFG) and toggle the normal flag through there.
    The normal mode flag was left out in the Demo 4 CFG due to the SRB2 Formatinterface defaulting normal mode to "Flip".
